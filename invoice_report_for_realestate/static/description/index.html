OSUS Invoice Report Odoo 17

Professional UAE Tax Invoice and Bill templates with smart dispatch and real estate enhancements. Includes portal-friendly QR codes, commission fields, and improved payment voucher summaries.

Key Features

  • Default print overrides for customer invoices and vendor bills
  • Real estate fields: sale value, project, unit, buyer, deal tracking
  • QR code generation with secure portal links
  • Amount-in-words and clean report styling

Dashboard Integration

Adds helpful list-view columns for property operations. Quantity column shows the sum of invoice line quantities.
