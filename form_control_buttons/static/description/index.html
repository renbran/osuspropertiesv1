Overview

The Save Button Text module enhances the Odoo user experience by adding clear Save and Discard buttons to the form view of any installed module. This helps users easily save their changes or discard them, improving usability and reducing accidental data loss.

How It Works

  1. Install the Save Button Text module from the Apps menu.
  2. Open any form view in Odoo (e.g., Contacts, Sales Orders, etc.).
  3. You will see Save and Discard buttons at the bottom of the form.
  4. Click Save to store your changes, or Discard to revert unsaved changes.

Key Features

  • Adds Save and Discard buttons to all form views
  • Improves user experience and clarity
  • Reduces accidental data loss
  • Works with any installed module
