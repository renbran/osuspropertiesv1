üì¶ Installation

Quick Install

  1. Download and extract the module to your Odoo addons directory
  2. Restart your Odoo server
  3. Go to Apps menu and click Update Apps List
  4. Search for "Announcement Banner"
  5. Click Install

Configuration

After installation, navigate to Announcements menu (visible to System Administrators only). Create your first announcement with a title and message, set scheduling options, and activate it!
